# 问题卡模板

每个问题整理成一张卡，后续再扩展成录屏口述稿。

## 模板

```text
标题：

现场现象：

很多人容易误判的地方：

可能原因：

判断顺序：

屏幕上能展示的操作：

最后怎么验证：

适合口述的句子：
```

## 示例

```text
标题：
同一个共享目录，有的电脑能打开，有的打不开

现场现象：
共享目录本身能用，但不是每台电脑都能打开。

很多人容易误判的地方：
一上来就改共享权限，或者反复重输账号密码。

可能原因：
访问电脑保存了旧凭据，网络类型不对，防火墙策略不同，或者系统版本差异导致表现不一样。

判断顺序：
先确认共享目录在本地是否正常，再确认访问电脑能不能到达目标 IP，再看凭据、防火墙和网络类型。

屏幕上能展示的操作：
共享目录、网络类型、凭据管理器、防火墙、用电脑名和 IP 分别访问。

最后怎么验证：
换两台电脑连续访问，打开文件并保存一次。

适合口述的句子：
这个地方很多人会以为是共享没有设置好，其实不一定。你看这一台能进，说明共享本身大概率没问题，那就要看访问这台电脑自己的状态。
```
