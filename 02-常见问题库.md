# 常见问题库

这些问题不是按软件功能整理，而是按真实使用时容易卡住的地方整理。

## 一、Windows 共享

1. 共享设置明明很简单，为什么用几天以后又打不开？
2. 同一个文件夹，为什么这台电脑能进，那台电脑进不去？
3. 为什么输入账号密码还是提示不对？
4. 为什么系统更新以后共享突然不能用了？
5. 为什么网络发现开了，还是看不到对方电脑？
6. 为什么用电脑名访问不行，换成 IP 又可以？
7. 为什么有些电脑会记住旧账号，一直连不上？
8. 为什么家庭版、专业版表现不一样？

## 二、异地访问

1. 文件夹在本地能打开，换个地方为什么打不开？
2. 两个地方都能上网，为什么共享目录还是访问不到？
3. 为什么不能直接把共享端口开放出去？
4. 为什么通道连上了，文件打开还是很慢？
5. 为什么远程系统能打开，但文件复制很卡？
6. 为什么换了路由器以后，原来的访问方式失效？
7. 为什么 IP 段冲突会导致看起来通了但实际不稳定？
8. 为什么只测一次打开不能说明问题？

## 三、网关和维护

1. 为什么电脑多了以后，不适合每台都单独装连接工具？
2. 为什么固定一台设备做网关，维护会简单很多？
3. 网关电脑关机以后，其他电脑为什么都访问不到？
4. 为什么网关要固定 IP？
5. 为什么网络通道正常，但某台电脑仍然打不开？
6. 为什么要区分线路问题和单台电脑问题？

## 四、同步和备份

1. 同步和备份为什么不是一回事？
2. 文件同步了，误删以后为什么另一边也没了？
3. 只同步指定目录时，为什么不能直接整盘同步？
4. 同步状态显示正常，为什么还是要实际打开文件测试？
5. 为什么要测试新建、修改、删除、恢复这几个动作？
6. 大文件同步为什么要看速度，也要看中途会不会断？

## 五、状态验证

1. 调完以后为什么不能只看能不能打开？
2. 为什么要连续打开几次目录？
3. 为什么要测试保存以后另一边能不能刷新出来？
4. 为什么要测试大一点的文件复制速度？
5. 为什么要点几个常用页面，而不是只看登录页？
6. 为什么要留意断线重连和长时间运行状态？
