# 案例 002：同一个共享有的电脑能上有的打不开

## 主题

共享目录本身能用，不代表每台访问电脑状态都正常。

## 口述稿

你看这个共享文件夹，如果这一台电脑能打开，说明这个共享本身大概率是没有完全坏掉的。

但是另外一台电脑打不开，这个时候不要马上去改共享权限。

很多人一看到打不开，就一直在共享设置里面改来改去。其实方向可能已经偏了。

这个时候更应该看打不开的那台电脑。

比如它是不是保存过旧的账号密码。Windows 有时候会自动拿以前的凭据去连，你这边明明输入新的，它还是走旧的，所以就会一直提示不对。

还有网络类型也要看。有的电脑是专用网络，有的电脑是公用网络，防火墙策略就不一样。

所以我一般会先用 IP 访问一次，再看电脑名访问行不行。然后再去看凭据管理器，把旧的记录清掉。

这个顺序不要反。共享本身能被其他电脑打开，就先不要急着动它，先查打不开的那台电脑自己的状态。
