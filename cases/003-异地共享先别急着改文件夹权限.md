# 案例 003：异地共享先别急着改文件夹权限

## 主题

换了地方打不开共享，很多时候不是文件夹权限问题，而是网络本身没通。

## 口述稿

这个共享文件夹如果在同一个办公室里面用，其实设置对了以后一般就能打开。

但是换到另外一个地方，就不是单纯共享设置的问题了。

很多人会卡在这里，一直去改文件夹权限，改账号，改密码。其实两个地方的网络本来就没有通，你怎么改共享都没有用。

这个地方要先判断网络能不能到。

比如先看能不能访问到那个 IP，再看对应端口有没有通。只有网络通了以后，再去看共享权限才有意义。

如果顺序反了，就会一直在 Windows 共享里面绕。

所以异地访问这个问题，第一步不是点共享设置，而是先确认两边网络有没有真正打通。

网络通了以后，再回头测试共享目录，打开文件，保存文件，刷新另一边看状态。这样判断才比较准。
