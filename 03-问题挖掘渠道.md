# 问题挖掘渠道

## 1. 自己的客户资料

优先级最高。

可挖掘来源：

- 飞书多维表格里的订单备注。
- 可道云客户文件夹结构。
- 微信聊天记录。
- 客户售后问题。
- 部署交付记录。
- 客户续费前问过的问题。
- 客户不续费的原因。
- 远程协助时遇到的问题。

处理方法：

```text
原始客户问题
↓
翻译成老板关心的业务风险
↓
整理成问题卡
↓
生成短视频选题和销售话术
```

## 2. 官方文档

用于确认产品能力和边界。

可查方向：

- 群晖 Synology Drive。
- 群晖 ShareSync。
- 威联通 HBS。
- Windows SMB 文件共享。
- 可道云资料管理。
- VPN 远程访问。
- 勒索病毒防护和备份恢复。

官方文档适合发现：

- 同步不是备份。
- 权限不是只建文件夹。
- 远程访问不能随便公网暴露。
- 文件冲突会影响业务。
- 误删会同步到多地。
- 备份需要恢复测试。

## 3. 社区和论坛

用于发现真实用户抱怨。

可看渠道：

- 群晖社区。
- 威联通社区。
- Reddit r/synology。
- Reddit r/qnap。
- 知乎。
- V2EX。
- 什么值得买 NAS 评论区。
- B 站 NAS 视频评论区。
- 抖音评论区。
- 淘宝、闲鱼、服务商评价。

注意：社区里很多问题是技术用户提出的，需要改写成老板语言。

示例：

```text
技术问题：ShareSync 子目录不同步
老板问题：分公司拿不到最新客户资料，客户催交付时谁负责？
```

## 4. 同行服务商内容

可搜索关键词：

```text
企业文件共享解决方案
工厂文件管理方案
异地办公文件共享
图纸文件管理
企业资料防泄密
员工离职资料安全
NAS 数据备份方案
客户资料管理方案
```

观察重点：

- 同行在卖什么服务。
- 评论区客户在问什么。
- 哪些标题播放量高。
- 哪些问题容易引起老板焦虑。

## 5. 客户行业本身

实体工厂常见资料类型：

- 图纸。
- 报价单。
- 合同。
- 客户资料。
- 生产单。
- 送货单。
- 财务报表。
- 售后资料。
- 扫描件。
- 安装包和说明书。

行业问题要按业务影响表达，而不是按软件功能表达。
